# Ariadne Brand Notes

## Naming rules

- Product name: `Ariadne`
- GitHub repo name: `ariadne-doc-assistant`
- Python package/import path: `ariadne_doc_assistant`
- CLI/docs command name: `ariadne-doc`

Use `Ariadne` in README text, documentation prose, and any future UI text. Use `ariadne_doc_assistant` only for Python imports and module paths.

## Logo

- `assets/ariadne-logo-dark.png` is the preferred logo variant for light backgrounds.
- `assets/ariadne-logo-light.png` is the preferred logo variant for dark backgrounds.
- `assets/ariadne-logo.png` remains available as a neutral fallback asset.
- The current mark uses an original yarn-ball/thread motif derived from the idea of Ariadne's thread.
- The logo was created for this project and may be used with the repository under the same open-source distribution context as the rest of the repo.
- The dark and light logo variants are the primary UI assets for theme-aware surfaces.

## Suggested colors

- Deep teal: `#0F4C5C`
- Warm orange: `#E36414`
- Soft sand background: `#F5F1E8`
- Dark text: `#14213D`
