## Naming matrix

- Product name: `Ariadne`
- GitHub repo name: `ariadne-doc-assistant`
- Python module/package: `ariadne_doc_assistant`
- CLI/docs command name: `ariadne-doc`

## Why the Python module differs

The human-facing product name is **Ariadne**, but the Python import path is **`ariadne_doc_assistant`** because the package name `ariadne` already exists in the Python ecosystem for a GraphQL library. Using `ariadne_doc_assistant` avoids ambiguity and accidental dependency conflicts.

## Usage rule

Use `Ariadne` in product-facing text, thesis materials, and future UI strings. Use `ariadne_doc_assistant` only in Python code, package metadata, imports, and Uvicorn module paths.
